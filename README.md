# Frasta

## Configuration
* create virtual environment:
`python -m venv .venv`

* activate:
`.venv\Scripts\activate.bat`

* instal packages:
`.venv\Scripts\pip.exe install -r requirements.txt`

* generating of requirements.txt:
`.venv\Scripts\pip.exe freeze > requirements.txt`
